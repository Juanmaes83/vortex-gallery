# Vortex Gallery

Galeria de arte interactiva con vortex WebGL 3D, bilingue (ES/EN), 8 paginas navegables y estrategia de conversion completa.

## Stack tecnologico

- React 19 + TypeScript + Vite
- Three.js (vortex WebGL)
- Tailwind CSS + shadcn/ui
- GSAP + ScrollTrigger
- react-i18next (bilingue)
- Swiper (carruseles)

## Desarrollo local

```bash
npm install
npm run dev
```

## Build para produccion

```bash
npm run build
```

Los archivos estaticos se generan en la carpeta `dist/`.
